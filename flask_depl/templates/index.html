Project Overview

Machine Learning is found almost everywhere. The healthcare industry is not an exclusion.
It plays a significant role in predicting the presence/absence of diseases. Identification of such information well in advance can
provide significant insights to doctors for effective treatments.

In this project, a web application is designed to integrate efficient models for many diseases trained on rigorous datasets
using advanced Machine Learning and Deep Learning Algorithms.
The user-friendly interface enables users to submit clear and concise symptom-based clinical findings of diagnosis
in text and image formats and obtain predictions.

This system aids in a no-contact treatment scheme minimizing the risks for both doctors and patients, which is crucial in current times.
It acts as a catalyst to speed up the treatment process and can enable regular and timely medical checkups for the underprivileged.

Advantages

1) Many of the currently existing systems use various models for disease prediction separately.
Multiple disease prediction on a single interface is not yet fully employed.

2) Availability of basic precautions and disease metadata helps to raise awareness among users.
This feature was not often included in existing models.

3) Fast, simple and secure

4) Easy to use

5) Medical terminology is clearly explained

6) Accurate

How to Use

Pretty simple!

1) Just select the disease which you want to obtain prediction for from the list

2) Upload the image or fill the form (by refering to your clinical report

3) Thats it! View your results on the same page

4) Find the detailed information about the disease on the information page
